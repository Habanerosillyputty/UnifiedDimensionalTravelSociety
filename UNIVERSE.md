# Unified Dimensional Travel Society Universe

This repository contains the fictional universe created with Copilot, including the wormhole portal graphic and supporting README content.

## Universe Concept
The Unified Dimensional Travel Society is a secretive alliance exploring alternate dimensions via wormhole transit. Their motto is "The New World is through a Wormhole," and they use advanced portal technology to cross boundaries and discover new realities.
